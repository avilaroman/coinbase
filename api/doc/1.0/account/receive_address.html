GET /api/v1/account/receive_address
Get the user's current bitcoin receive address.

Authenticated resource that returns the user’s current bitcoin receive address. This can be used to generate scannable QR codes in the bitcoin URI format or to send the receive address to other users.

Examples

# Request
GET https://api.coinbase.com/v1/account/receive_address
# Response
{
  "success": true,
  "address": "muVu2JZo8PbewBHRp6bpqFvVD87qvqEHWA",
  "callback_url": null
}

Params

Param name Description
account_id
optional

Specify for which account we fetch the receive address. The default is your primary account.


Value: Must be String
