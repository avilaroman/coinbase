GET /api/v1/prices/sell
Get the total sell price for some bitcoin amount.

Unauthenticated resource that tells you the total amount you can get if you sell some bitcoin. The default quantity is 1 BTC.

The price is broken down as follows:

  1. subtotal - this is the current market price to sel this quantity of bitcoin, we integrate with a number of exchanges on the backend to calculate this price
  2. fees - an array of fees you will pay if you wish to sell at this price through Coinbase
  3. total - the subtotal minus fees, this is the total amount you will receive if you wish to sell this quantity on Coinbase

Note that exchange rates fluctuate so you will see different numbers than in the example below. The price per bitcoin will also decrease for larger purchases (there is a ‘depth’ to the market, so only a limited number of bitcoins can be sold at the best price).

You will also see amount and currency fields at the root level which duplicate the total field (not shown in the example below). These are legacy fields left in the API to maintain backwards compatibility with a previous version.

Examples

# Request
GET https://api.coinbase.com/v1/prices/sell
{
  "qty": 1
}

# Response
{
  "subtotal": {
    "amount": "9.90",
    "currency": "USD"
  },
  "fees": [
    {
      "coinbase": {
        "amount": "0.10",
        "currency": "USD"
      }
    },
    {
      "bank": {
        "amount": "0.15",
        "currency": "USD"
      }
    }
  ],
  "total": {
    "amount": "9.65",
    "currency": "USD"
  },
  "amount": "9.65",
  "currency": "USD"
}

Params

Param name Description
qty
optional

The quantity of bitcoin you would like to sell (default is 1).


Value: Must be Float
currency
optional

Default is USD. Right now this is the only value allowed.


Value: Must be String
