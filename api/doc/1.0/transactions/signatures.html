PUT /api/v1/transactions/:id/signatures
Submit required signatures for a multisig spend transaction.

Examples

# Request
PUT https://api.coinbase.com/v1/transactions/53f3d9e0cbf034354a000132/sighashes
{

    "signatures": [

    ]
  }]
}

Params

Param name Description
signatures
required

Array of inputs with array of signatures each


Value: Must be Array
signatures[][index]
required

Input index that you got in the sighash call


Value: Must be String
signatures[][signatures]
required

Array of signatures produced by signing the sighash for this input with your private keys


Value: Must be Array
