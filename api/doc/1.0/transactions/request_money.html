POST /api/v1/transactions/request_money
Send an invoice/money request to an email address.

Authenticated resource which lets the user request money from a bitcoin address.

If you pass an amount param it will be interpreted as a bitcoin amount. Alternatively you can pass an amount_string and amount_currency_iso such as ‘USD’ or ‘EUR’ and the equivalent amount of bitcoin will be sent at current exchange rates.

Examples

# Request
POST https://api.coinbase.com/v1/transactions/request_money
{
  "transaction": {
    "from": "user1@example.com",
    "amount": "1.234",
    "notes": "Sample transaction for you"
  }
}

    "id": "501a3554f8182b2754000003",
    "created_at": "2012-08-02T01:07:48-07:00",
    "hsh": null,
    "notes": "Sample request for you!",
    "amount": {
      "amount": "1.23400000",
      "currency": "BTC"
    },
    "request": true,
    "status": "pending",
    "sender": {
      "id": "5011f33df8182b142400000a",
      "name": "User One",
      "email": "user1@example.com"

      "id": "5011f33df8182b142400000e",

# Request
POST https://api.coinbase.com/v1/transactions/request_money
{
  "transaction": {
    "from": "not an email address",
    "amount": "1.234",
    "notes": "Sample transaction for you"
  }
}
# Response
{
  "success": false,
  "errors": [
    "Please enter an email address in the from field."
  ],
  "transaction": {...}
}

# Request
POST https://api.coinbase.com/v1/transactions/request_money
{
  "transaction": {
    "from": "user1@example.com",
    "amount": "-1",
    "notes": "Sample transaction for you"
  }
}
# Response
{
  "success": false,
  "errors": [
    "You must enter a valid amount"
  ],
  "transaction": {...}
}

Errors

401 Unauthorized

Params

Param name Description
account_id
optional

Specify which account will be credited. The default is your primary account.


Value: Must be String
transaction
required

Transaction object containing the params


Value: Must be a Hash
transaction[from]
required

An email address to send the request


Value: Must be String
transaction[amount]
optional

A string amount that will be converted to BTC, such as ‘1’ or ‘1.234567’. Also must be >= ‘0.01’ or it will shown an error. If you want to use a different currency you can set amount_string and amount_currency_iso instead. This will automatically convert the amount to BTC and that converted amount will show in the transaction.


Value: Must be String
transaction[amount_string]
optional

A string amount that can be in any currency. If you use this with amount_currency_iso you should leave amount blank.


Value: Must be String
transaction[amount_currency_iso]
optional

A currency symbol such as USD, EUR, or ‘BTC’


Value: Must be String
transaction[notes]
optional

Optional notes field. Included in the email that the recipient receives.


Value: Must be String
