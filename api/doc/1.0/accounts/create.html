POST /api/v1/accounts
Create an account.

Authenticated resource that will create a new account for the user.

Examples

# Request
POST https://api.coinbase.com/v1/accounts
{
  "account": {
    "name": "Savings Wallet"
  }
}

# Request
POST https://api.coinbase.com/v1/accounts
{
  "account": {
    "name": "Multisig Wallet",
    "type": "multisig",
    "m": 2,
    "n": 3, # Here you just announce that this account will have 3 keypairs, but then you only provide the 1st one. The rest of the keys can be provided through the /keypairs call.
    "xpubkeys": [
      "xpub661MyMwAqRbcFo8WEPnst2sE8MTLe9DszR7eYhtkVuiUskpAggETvYQeSBWTuwoxZrZvf18w75AzfjLhzihWGagvcMa4J9nDWjmiD2UrAEF"
    ]
  }
}

Params

Param name Description
account
required

Account object containing the params


Value: Must be a Hash
account[name]
required

Account name


Value: Must be String
account[type]
optional

Type of account. If creating a HDM account, specify ‘multisig’, otherwise leave blank.


Value: Must be String
account[m]
optional

Number of required signatures to spend from a HDM multisig account.


Value: Must be Integer
account[n]
optional

Number of keypairs that will comprise the complete account, when creating the account in stages. You need to pass this parameter if you will provide less keys than ‘m’.


Value: Must be Integer
account[xpubkeys]
optional

Array of extended public keys of BIP32 wallets. Length of array must be greater or equal than ‘m’ to create a complete account. Otherwise you should provide ‘n’.


Value: Must be Array
