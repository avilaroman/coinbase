Addresses
List bitcoin addresses associated with your account.

>>>

GET /api/v1/addresses
List bitcoin addresses associated with this account.

Authenticated resource that returns bitcoin addresses a user has associated with their account.

Bitcoin addresses remain associated with an account forever, and can be used indefinitely. Sometimes it can be useful for a user to go back and see all addresses in their account.

By default 25 addresses are returned at a time. You can set the page and limit parameter to adjust how pagination works.

Note: for accounts created using the API call POST /api/v1/accounts, an address needs to be generated on the account before it will be returned on this page. An address can be generated either by loading up the accounts page or by using the API call POST /api/v1/account/generate_receive_address

Params

Param name Description
page
optional

Can be used to page through results. Default value is 1.


Value: Must be Integer
limit
optional

Number of records to return. Maximum is 1000. Default value is 25.


Value: Must be Integer
account_id
optional

Specify which account is used for fetching data. The default is your primary account.


Value: Must be String
query
optional

Optional string match to filter addresses. Matches the address itself and also if the use has set a ‘label’ on the address.


Value: Must be String
