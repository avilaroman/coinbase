GET /api/v1/transfers
List a user's recent buys and sells.

Authenticated resource which returns the user’s bitcoin purchases and sells. Sorted by created_at in descending order.

Status can be Pending, Completed, Canceled, or Reversed.

The transaction_id can be used to link a transfer to a transaction.

By default 25 transfers are returned at a time. You can set the page and limit parameter to adjust how pagination works.

Examples

# Request
GET https://api.coinbase.com/v1/transfers

Params

Param name Description
account_id
optional

Specify which account is used for fetching data. The default is your primary account.


Value: Must be String
page
optional

Can be used to page through results. Default is 1.


Value: Must be Integer
limit
optional

Number of records to return. Maximum is 1000. Default is 25.


Value: Must be Integer
