Params

Param name Description

optional

Can be used to page through results. Default value is 1.

optional

optional

Value: Must be Boolean

optional

Value: Must be String

optional

Number of required signatures to spend from a HDM multisig account.


Value: Must be Integer

optional

Value: Must be Array

POST /api/v1/accounts/:account_id/primary
Set account as primary.

Authenticated resource that lets you toggle the primary status on a specific account. You must pass the :account_id of the account in the url. This call will behave as follows:

  • If the specified account is already primary, it will mark it as non-primary and create a new primary wallet.
  • If the specified account is not primary, it will mark it as primary and set the previously-primary account as non-primary.

Examples

# Request
POST https://api.coinbase.com/v1/accounts/53752d3e46cd93c93c00000c/primary

# Response
{
  "success": true,
}
